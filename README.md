
# sequential-logic-circuits-state-reduction

A C++-based implementation of finite state machine (FSM) minimization using the implication table method, developed as part of a Digital Logic II project

## How to run:

clone this repo and cd into the directory and run CMAKE
`` qmake6 StateMachineApp.pro && make -j4 ``

then after it finishes compiling  

``./StateMachineMinimizer``
